#  Real Estate Price Forecasting & Risk Management

A Data Science project predicting real estate prices and quantifying financial risk using Machine Learning, Time Series, and Monte Carlo simulation.

---

##  Results

| Model | MAE | R² |
|-------|-----|----|
| Ridge | $16,209 | 0.864 |
| Random Forest | $16,629 | 0.842 |
| **XGBoost ** | **$14,408** | **0.877** |

---

##  Project Structure

| File | Description |
|------|-------------|
| `01_data_preparation.py` | Data loading & cleaning |
| `02_eda.py` | Exploratory Data Analysis |
| `03_ml_modelisation.py` | ML models + SHAP explainability |
| `04_series_temporelles.py` | SARIMA + Prophet forecasting |
| `05_risk_management.py` | Monte Carlo, VaR, Stress Testing |
